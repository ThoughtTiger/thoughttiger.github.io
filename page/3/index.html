git+github+hexo+MarkdownPad创建自己博客系统

1 准备


1.1 安装软件

软件 作用
git for windows 执行git命令
node.js for windows 安装hexo的前提
hexo 一个简单、快速、强大的博客发布工具
MarkdownPAD 编写md文件的工具


1.1.1 安装git for windows

https://git-for-windows.github.io/


1.1.2 安装nodejs for windows

https://nodejs.org/en/


1.1.3 安装hexo

安装好nodejs后，系统会自带npm命令。
安装hexo
npm install -g hexo
这里有warn可以不用管
